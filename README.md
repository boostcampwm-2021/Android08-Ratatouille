## ✅ 앱 배포 로그

### v1.1

- 레시피 추가 페이지 동작 구현

  - 드래그와 스와이프로 단계 순서 변경 및 제거
  - 레시피 등록하기 전 유효성 검사
  - 레시피 저장 구현
  - 저장 후 팝업 알림 구현

- 내 레시피 페이지 UI

- 힐끔보기 페이지 UI

### v2.0

- 내 레시피 화면
  - 라디오 버튼 클릭으로 아이템 정렬 및 즐겨찾기에 등록된 레시피만 화면에 표시
  - 레시피 1회 클릭시 소요시간 재료 표시
  - 레시피 2회 클릭시 레시피 개요 화면으로 이동
  - 스크롤 다운시 화면 새로고침
  - 스크롤 업시 데이터 페이징
  - Room DB에서 데이터 로드 실패시 snack로 데이터 로드 실패 알림 및 새로고침
  - 디바이스 가로길이에 맞춰 SpanCount 동적으로 지정
  - Flow를 이용하여 레시피 추가/삭제/수정 시 화면에 바로 반영

- 레시피 추가 화면
  - 누락된 정보 확인 시 스크롤 이동

- 레시피 검색 화면
  - 네트워크 / 로컬 검색 기능
  - 레시피 클릭 시 개요 화면 이동

- 힐끔보기 화면
  - 라디오 버튼 클릭시 데이터 불러ㅇ
  - 스크롤 내릴 시 페이징 되면서 리스느 표시
  - 레시피 클릭 시 개요 화면으로 이동
  - 네트워크 연결 끊어 졌을시 스낵바로 알림

- 레시피 개요 화면
  - 플로팅 버튼 구현(애니메이션 추후 구현 예정)
  - 즐겨찾기 등록 / 해제
  - 업로드 구현
  - 수정버튼 클릭 시 레시피 수정 페이지 이동
  - 훔쳐오기 구현
  - 원격 / 로컬 삭제 구현

- 레시피 상세 화면
  - 알람 종료시 사운드로 알림 구현
  - 앱을 나가도 알람 계속 진행되게 구현
  - 알람 애니메이션 구현

### v2.1

- 동일한 레시피를 여러 번 훔쳐오면 그 중 한 개만 정상 동작하는 오류 수정
- 로딩, 확인 화면 다른 곳 누르면 취소되는 문제 수정
- 레시피와 첫 번째 단계에 이미지 설정하면 저장이 이상하게 되는 오류 수정
- 수정 및 업로드 이슈 해결
- 내 레시피 화면 프로그래스바 수정

### v2.2

- 시간 입력 후 다음으로 넘어갈 시 Crash 나는 현상 수정
- 플로팅버튼 클릭 시 애니메이션 발생 추가

### v3.0

- Theme 색상 변경
- 알람 Notification 기능 추가
- WorkManager를 통한 자동 업로드 제시도
- Background Process Kill 대응
- 디바이스 화면 가로 길이에 맞춰 대응
- 디자인 업데이트
